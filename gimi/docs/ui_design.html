UI设计规范说明

🎨 视觉风格

  • 风格定位：简约现代、玻璃拟态（Glassmorphism）
  • 背景：实时相机预览背景，半透明黑色渐变遮罩
  • 控件：毛玻璃效果，圆形按钮，半透明白色背景

🎯 核心交互

  • 麦克风：点击/长按开始录音，显示声波动画
  • 取消按钮：红色醒目，中止当前输入
  • 相机按钮：点击弹出拍照/选图菜单
  • 声波动画：三个圆点呼吸动画，模拟音量波动

📐 布局规范

  • 顶部状态栏 + 导航栏（50px + 50px）
  • 底部控制栏高度 160px，适配安全区域
  • 按钮直径 72px，间距 30px
  • 支持 iPhone 全面屏（390×844）

🎨 色彩方案

✨ 状态说明

点击顶部按钮可切换不同状态预览：待机 → 录音中 → 识别中 → 图片预览

* 本原型仅展示UI效果，不含真实业务逻辑
