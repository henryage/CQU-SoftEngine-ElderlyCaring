# 老人端接口文档

> 模糊视觉辅助问答系统 — 老人端微信小程序  
> 服务地址：`http://10.242.5.159:8090` | Swagger：`http://10.242.5.159:8090/docs`

---

## 1. 基础约定

### 1.1 统一响应格式

```json
{"code": 0, "msg": "ok", "data": {...}}
```

### 1.2 鉴权

除 `/auth/wx-login` 外，所有接口 Header 携带 `Authorization: Bearer <token>`。token 由登录返回，30 分钟有效。

### 1.3 错误码

| HTTP | 含义 | 常见原因 |
|:----:|------|---------|
| 200 | 成功 | — |
| 400 | 参数错误 | 必填缺失 |
| 401 | 未认证 | token 过期 |
| 403 | 权限不足 | 非老人端 token |
| 404 | 不存在 | 资源未找到 |
| 500 | 服务器错误 | 大模型/DB异常 |

## 3. 接口详情

### 3.1 登录 `POST /auth/wx-login`

```json
// 请求
{"code": "my_test_code", "user_type": "user"}
// 响应
{"code":0,"data":{"token":"eyJ...","refresh_token":"eyJ...","token_type":"Bearer","user_type":"user","ref_id":6,"nickname":"老人6"}}
```

**dev 模式**：同一 code 永远登录同一老人。

### 3.2 我的信息 `GET /auth/me`

```json
{"code":0,"data":{"ref_id":6,"user_type":"user","is_child":false,"is_elder":true}}
```

> `ref_id` 即老人 ID，可将此数字告知子女完成绑定。

### 3.3 修改昵称 `PUT /auth/nickname?nickname=妈妈`

```json
{"code":0,"msg":"昵称已更新","data":{"user_id":6,"old":"老人6","new":"妈妈"}}
```

### 3.4 生成绑定验证码 `POST /auth/generate-bind-code`

```json
{"code":0,"data":{"code":"385721","expires_in_seconds":300,"user_id":6}}
```

> 将 6 位验证码告知子女，子女凭此完成绑定，5 分钟有效。

### 3.5 心跳 `POST /auth/heartbeat`

保持在线状态，30 秒调一次。无需 body。

### 3.6 拍照上传 `POST /media/upload/image`

FormData 上传 file 字段。返回 `{"file_url":"/files/img_xxx.jpg","enhanced_url":"/files/img_xxx_enhanced.jpg"}`。

### 3.7 录音上传 `POST /media/upload/voice`

同上。返回 `{"file_url":...,"asr_text":"...语音转写结果..."}`。

### 3.8 智能问答 `POST /qa/ask`

| 参数 | 说明 |
|------|------|
| input_type | text / image / voice |
| text | 问题文本（图片/语音场景可选） |
| media_url | 图片用 `enhanced_url`，语音用 `file_url` |

```json
// 文本
{"input_type":"text","text":"布洛芬怎么吃"}
// 图片（先上传拿到 enhanced_url）
{"input_type":"image","text":"这是什么药","media_url":"/files/img_xxx_enhanced.jpg"}
// 语音（先上传拿到 asr_text）
{"input_type":"text","text":"今天血压多少正常"}
```

返回：`{"answer":"...","session_id":...,"msg_id":...,"intercepted":false}`

### 3.9 历史 `GET /qa/history?page=1&page_size=20`

### 3.10 紧急呼叫 `POST /alert/emergency/call`

无需 body。后端写预警 → 推送所有绑定子女。

### 3.11 用药提醒 `GET /reminder/medication`

```json
{"data":[{"reminder_id":1,"drug_name":"硝苯地平","dosage":"1片","remind_time":"08:00"}]}
```

## 4. 核心功能流程

### 4.1 登录与身份

```
wx.login()
  → POST /auth/wx-login {code, user_type:"user"}
  → 存 token + ref_id
  → onShow: POST /auth/heartbeat（每30秒）
```

### 4.2 文本问答

```
老人输入"血压多少正常"
  → POST /qa/ask {input_type:"text", text:"血压多少正常"}
  → 小猫"思考"动画
  → 收到 {"answer":"正常血压是...","intercepted":false}
  → 展示回答 + 语音播报
```

### 4.3 拍照问答

```
点击拍照 → 选择图片
  → POST /media/upload/image (FormData{file})
  → 拿到 enhanced_url
  → POST /qa/ask {input_type:"image", text:"这是什么药", media_url: enhanced_url}
  → 小猫"思考"动画（可能要等 20-40 秒，VLM 推理图片）
  → 展示回答
```

### 4.4 语音问答

```
按住"说话"按钮
  → 录制 → .stop()
  → POST /media/upload/voice
  → 拿到 asr_text
  → 吐司展示 asr_text 让老人确认
  → POST /qa/ask {input_type:"text", text: asr_text}
  → 展示回答
```

### 4.5 绑定子女

```
老人：POST /auth/generate-bind-code → 拿到 6 位验证码 "385721"
  → 告知子女验证码
子女：POST /child/bind {"code":"385721","relation":"母亲"}
  → 绑定成功
```

### 4.6 紧急呼叫

```
老人点击"紧急"按钮
  → POST /alert/emergency/call
  → 后端写预警 → 推送所有绑定子女订阅消息
  → 前端展示"已通知子女"
```

## 5. 前端实现要点

```javascript
// 登录
wx.login({ success: res => {
  wx.request({ url: BASE+'/auth/wx-login', method:'POST',
    data: {code:res.code, user_type:'user'},
    success: r => { wx.setStorageSync('token', r.data.data.token) }
  })
}})

// 查看我的 ID（"我的"页面展示给老人）
wx.request({ url: BASE+'/auth/me', header:{Authorization:'Bearer '+token},
  success: r => this.setData({ userId: r.data.data.ref_id })
})

// 心跳（每30秒）
setInterval(() => wx.request({
  url: BASE+'/auth/heartbeat', method:'POST',
  header: {Authorization:'Bearer '+token}
}), 30000)
```
